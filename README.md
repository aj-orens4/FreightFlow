Download n'yo 'to tapos extract n'yo 'yung file.
After niyan, punta kaso VS Code

Punta kayo sa File, located siya sa upper left corner ng VS Code, tapos selec n'yo 'yung file na in-extract n'yo

After that punta naman kayo sa upper left corner ulit ng VS Code, hanapin n'yo 'yung terminal

Click n'yo tapos Click "New Terminal"
('pag 'di nag work 'to, punta kayo sa baba, kung saan nakalagay 'yung output, hanapin n'yo 'yung (+) ktabi no'n 'yung pindutin n'yo tapos click n'yo cmd (Command Prompt)

after niyan type n'yo "cd freightflow" tapos enter

after ulit niyan, type n'yo ulit "cd server" tapos enter ulit

after niyan type n'yo "npm install" tapos enter ulit

after and last step is type nyo ulit "npm start"

okay na 'yan, nag run na 'yung system.

punta na lang kayo sa link na 'to:

http://localhost:4000
